# 黑盒圣经 · 终极合集 & 皮特弹幕

### 启动流程🚀

pnpm i

pnpm run docs:dev

### 贡献指南🎨

黑盒圣经：黑盒圣经路径下添加图片文件+md文件，然后在.vitepress/config.mts添加目录

注：如果原帖的链接还可以访问，请在 commit message 中添加原帖子链接

皮特弹幕：pite/index文件中直接添加即可，无需添加目录

最后 pnpm run docs:build
